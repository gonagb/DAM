Etiqueta <section>

La etiqueta <section> se utiliza para dividir el contenido en secciones temáticas.

Etiqueta <article>

La etiqueta <article> se utiliza para representar un contenido independiente y autónomo, como una noticia o un artículo.

Etiqueta <footer>

La etiqueta <footer> se utiliza para definir el pie de página de una sección o página web.
